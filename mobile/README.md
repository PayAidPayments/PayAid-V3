# PayAid V3 Mobile App

React Native mobile application for PayAid V3.

## Setup

```bash
cd mobile
npm install
```

## Development

```bash
# iOS
npm run ios

# Android
npm run android
```

## Features

- Core CRM (contacts, deals, tasks)
- Order tracking
- Attendance (check-in/out)
- Invoice viewing
- Dashboard with key metrics

## API Integration

The mobile app connects to the same backend API:
- Base URL: `https://api.payaid.com` (production)
- Authentication: JWT tokens
- All API endpoints are tenant-aware

## Status

Mobile app structure and API integration points are defined. Full implementation requires React Native development setup.
