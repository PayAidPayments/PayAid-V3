## ðŸ“– Training Modules

### Module 1: Email Integration (30 min)

**Learning Objectives:**
- Connect email accounts
- Understand two-way sync
- Use email tracking
- Compose emails from CRM

**Exercises:**
1. Connect your Gmail/Outlook account
2. Send a test email and verify it's logged
3. View email open/click tracking
4. Compose email from a contact page

**Quiz:**
- What is two-way email sync?
- How do you track email opens?
- Where are email attachments stored?

---

### Module 2: AI Lead Scoring (45 min)

**Learning Objectives:**
- Understand lead scoring (0-100)
- Interpret lead scores
- Set up auto-routing
- Use lead qualification workflow

**Exercises:**
1. Review lead scores for 10 contacts
2. Identify top 5 hot leads
3. Set up auto-routing rule
4. Create nurture sequence for low-scored leads

**Quiz:**
- What factors affect lead score?
- What score indicates a hot lead?
- How do you route leads automatically?

---

### Module 3: Deal Management (40 min)

**Learning Objectives:**
- Manage deal pipeline
- Use deal rot detection
- Understand closure probability
- Forecast revenue

**Exercises:**
1. Create 5 test deals
2. Move deals through stages
3. Review deal rot dashboard
4. Check closure probability for a deal
5. View revenue forecast

**Quiz:**
- What is deal rot?
- How is closure probability calculated?
- What affects revenue forecast accuracy?

---

### Module 4: Workflow Automation (35 min)

**Learning Objectives:**
- Create automation workflows
- Set up triggers and actions
- Test automations
- Monitor automation performance

**Exercises:**
1. Create "Hot Lead Auto-Assign" workflow
2. Create "Deal Stage Change" workflow
3. Create "Follow-up Email" sequence
4. Test each automation

**Quiz:**
- What are the available trigger types?
- How do you test an automation?
- Can automations be conditional?

---

### Module 5: Mobile App (25 min)

**Learning Objectives:**
- Use mobile app features
- Work offline
- Use voice commands
- Quick capture features

**Exercises:**
1. Download and set up mobile app
2. Test offline mode
3. Use voice command: "Show my top 3 deals"
4. Scan a business card (or test with sample)
5. Record a voice note

**Quiz:**
- How does offline sync work?
- What voice commands are available?
- How do you swipe to change deal stage?

## ðŸ“š Quick Reference Guides

### Keyboard Shortcuts

- `Ctrl/Cmd + K` - Quick search
- `Ctrl/Cmd + N` - New contact/deal
- `Ctrl/Cmd + /` - Command palette
- `Esc` - Close modal/dialog

### Common Tasks

**Add Contact:**
1. Click "New Contact"
2. Fill in details
3. Save

**Create Deal:**
1. Go to Deals
2. Click "New Deal"
3. Link to contact
4. Set value and stage
5. Save

**Send Email:**
1. Open contact/deal
2. Click "Send Email"
3. Compose and send
4. Email auto-logged
